
# Kaggle---Ghouls-Goblins-and-Ghosts...Boo-
Kaggle Competition - Ghouls, Goblins and Ghosts...Boo!

Kaggle hosted the Ghouls, Goblins and Ghosts...Boo! competition in November 2016. This was my first participation in a kaggle competition. 

This was a classification algorithm to classify Ghouls, Goblins and Ghosts based on features like hair length, etc.
